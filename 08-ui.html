UI 개선안

발견 1(정보 부족이 구매를 가로막고 있음)은 밸런스 수치를 조정하지 않고 화면 개선만으로 상당 부분 해소됩니다. 아래는 그 목업입니다.

08-1

🖥️현재 UI 조건

기존 UI 구조 검수(37개 화면) 결과를 요약합니다.

유리텍스트 시스템이 TextMeshPro 100%, 주 폰트가 92% 단일화 → 전역 리스타일에 매우 유리한 조건입니다.
유리검수한 37개 화면 중 「변경 불가」로 판정된 화면이 0개입니다.
제약해상도 기준이 분리되어 있습니다. 로딩·메인은 1080×1920이나 전투 씬은 전부 800×600이며, 길드 아레나·PVP는 반응형이 아닙니다.
제약색상 35곳 이상이 로직에 고정되어 있어, 화면 데이터만 수정해서는 리스타일이 완결되지 않습니다.
제약전투 HUD가 8개 씬에 각각 복제되어 있어, HUD 변경 1건이 씬 5~8개 수정으로 이어집니다.
유저 진술과의 연결
게임 속 글자가 너무 작음. 처음에 할때는 너무 불편했다.
— 자유의견 [105]
아이패드 미니5는 UI가 많이 불편해요 ㅠㅠ
— 자유의견 [94]

두 진술 모두 해상도 기준 분리와 관련이 있습니다. 접수층의 86.2%가 30~40대인 게임에서 가독성은 선호의 문제가 아니라 접근성의 문제입니다.

08-2

📊강화 화면 — 성장 정보 표기

영웅 등급 레전더리(벨라)입니다. 영웅 강화(조각300개씩) 1레벨인데 20레벨까지 필요한 조각 갯수가 안나와서 엄두가 안납니다. 알아야 투자를 할 것 같습니다.
— 기타 게임제안 [20]
// 목업 · 좌: 현재 (총비용 확인 불가) / 우: 제안 (투자 판단 가능)
필요한 수치는 이미 존재합니다 (초월 300×20=6,000 / 각성 250×50=12,500 / 각성 효과 HP+2%·공격+3.5%·방어+1% per Lv → 데이터 룸). 다만 현재는 로직 상수로만 존재하여 화면에 표시할 수 없으므로, 데이터로 분리하는 작업이 선행됩니다.
이 화면 하나가 여는 것

밸런스 수치를 전혀 변경하지 않았습니다. 요구 조각도 6,000 그대로이고 효과도 그대로입니다. 표시만 추가하였습니다.

그러나 이제 유저는 「6,000조각이면 약 53,000원이니 투자할 것인가」를 판단할 수 있습니다. 투자 판단에는 총비용 예측이 전제됩니다. 현재는 그것이 불가능하여 시작 자체를 하지 않는 상태입니다.

동일한 논리가 벨트 옵션 최대수치에도 적용됩니다 — 유저 [제안 15]가 직접 요청한 사항입니다.

08-3

🎲확률 표기

법정 표시 의무 이행과 유저 신뢰 회복을 동시에 충족하는 화면입니다. 근거는 「밸런스 실측 · 확률 표시」를 참조해 주십시오.

// 목업 · 강화 화면 내 확률 표시 + 전체 확률표
⚠️ 선행 과제: 가챠는 현재 기록값과 실제 동작이 달라(04-5) 공개 이전에 확률 시스템을 정규화된 가중 테이블로 재구현하는 작업이 필요합니다. 강화 확률은 테이블이 이미 정상이므로 표시부터 가능합니다.
참고 — 국내 관행

리니지M은 확률 공개 의무화 대응으로 게임 내에서 확률을 「즉시 확인」할 수 있도록 UI를 개편하였습니다. 상점 뽑기팩, 합성 시도, 카드 개봉 시 구성 목록과 확률을 사전 확인할 수 있습니다.

넥슨은 유료 확률형 아이템의 실제 결과값을 일 단위로 자동 검증하고 이상탐지하며, 판매 첫날은 30분 단위로 집계합니다. 2025-08-01부터 입증책임이 게임사에 있으므로, 이러한 검증 체계는 법무 리스크 관리 항목이 되었습니다.

08-4

⏰던전 스케줄러

설문 4위(59표) + 30~40대 86.2%의 시간 제약에 대한 대응입니다. 사례는 나이트크로우 스케줄러입니다.

// 목업 · 던전 7종에 순차 자동 실행 적용
기존 던전 7종 진입 로직에 큐 레이어를 추가하는 구조입니다 — 신규 던전·신규 콘텐츠가 필요하지 않습니다. 원본 사례: 나이트크로우 스케줄러 (인벤)
이 항목을 우선 검토하는 이유

본 게임은 이미 던전 7종 + 출석 + 제작 + 미션으로 일일 루틴이 분산되어 있습니다. 유저는 이를 매일 수동으로 반복하여야 합니다.

「던전 7종을 순서·시간을 지정해 순차 자동 실행」 = 신규 콘텐츠 없이 체감 개선이 가장 큰 항목입니다.

그리고 이는 30~40대가 86.2%인 접수층의 시간 제약을 화면 설계로 직접 해소합니다. 저니 오브 모나크 사례가 시사하듯, 경쟁을 제거하는 대신 경쟁 참여 비용을 낮추는 방향에 해당합니다.

08-5

🏭길드 작업장

유저 요청 길드(2위, 68표) · 방치(4위, 59표) · 제작(6위, 45표)을 단일 시스템으로 대응하는 구성입니다.

// 목업 · 검은사막 일꾼 + RoK 2화폐 + AFK Arena 12시간 cap 결합
기부 소재는 기존 제작 재료를 재활용 = 신규 아이템 불필요합니다. 길드 코인은 기존 상점 9탭의 「길드」 탭에 부착됩니다. 참조: RoK 2화폐 · 검은사막 일꾼 · AFK Arena 12시간 cap
함께 검토할 항목

본 게임에는 유저 거래소가 없으므로, 제작 콘텐츠 활성화를 위해서는 기존 상점에 「제작품 납품」 탭(검은사막 황실 납품형)이 함께 필요합니다. 1일 상한을 길드 기여도에 비례시키면 인플레이션도 통제됩니다. → 업계 비교 · 제작
