콘텐츠 구성도

게임에 실제로 존재하는 콘텐츠를 코드 enum과 UI 오브젝트 기준으로 전수 정리했습니다. 아래 목록은 기획서가 아니라 빌드에 들어있는 것입니다.

하단 탭 — 게임의 뼈대

BottomTabType이 정의하는 10개 탭이 콘텐츠의 최상위 분류입니다.

스테이지

메인 진행은 5개 테마 × 3개 난이도입니다. TBL_STAGE는 50행이며, 1행 이름은 돼지의 성 I, 2행은 돼지의 성 II로 확인했습니다.

테마 (StageType)

Castle 성 Sea 바다 Desert 사막 Snow 눈 ToyLand 장난감나라

씬 오브젝트 StageMap_Castle_0 · _Sea_0 · _Desert_0 · _Snow_0 · _ToyLand_0로 실체 확인.

난이도 (StageDifficulty)

Normal Hard Hell

이벤트 스테이지 상품도 event_stage_normal / hard / hell로 동일하게 3분화됩니다.

던전 — 재화 수급의 핵심

DungeonType 8종. 이 중 4종만 던전 탭(DungeonTabType)에 노출되고, 나머지는 별도 진입점을 갖는 것으로 추정됩니다.

던전탭 노출성격 (추정)
Cave 동굴O보상형 — TBL_DUNGEON_CAVE_REWARD 존재
InfiniteTower 무한의 탑O층 무한 도전 — 전용 상품 2종(special/premium)
Colosseum 콜로세움O대전 콘텐츠 — ColosseumHero vs ColosseumMonster
SkyTemple 하늘신전O전용 맵 SkyTempleMap · DungeonDoorTemple
PigCow 돼지소-보스 던전 — PigCowKing/Queen
FishFight 물고기-Fish 4색 몬스터 던전
Miner 광부-채굴 연계 — MinerDungeon
Promotion 승급-TBL_ROYAL_PROMOTION 연계 승급전

모험 (Adventure)

AdventureType 5종 — CastleAndSea · Desert · PigCow · Snow · ToyLand. 스테이지 테마와 거의 겹치되 성과 바다가 하나로 묶이고 PigCow가 추가됩니다. 진행 조건은 AdventureConditionType으로 Level(레벨 도달) 또는 PrevIndex(선행 완료) 두 가지입니다.

시스템 전체 지도

RefreshEventType은 UI 갱신 이벤트 목록이라, 사실상 게임이 가진 시스템의 완전한 색인입니다. 38개 항목 전수:

Battle BattleState Currency Royal Equipment Quest Achievement User GuideMission Attendance Mine Bless Postbox Ranking Chat Artifact Pass Friend Book Youtube Dungeon Mole Setting Gacha Time Stat Hero Skill Rune Shop Tutorial Adventure Event RoyalBox Bingo DiceGame Collection AttendanceBuff

읽는 법 — 이 목록에 있는데 UI 탭에 없는 것(Book·Postbox·Ranking·Friend·Youtube·Mole·Bingo·DiceGame)은 탭이 아니라 팝업/서브 화면으로 진입하는 콘텐츠입니다. 실제로 UI_PostboxSlot, UI_RankingSlot, UI_FriendSlot, UI_YoutubeSlot, UI_MoleSlot, UI_DiceGameSlot 프리팹이 모두 존재합니다.
