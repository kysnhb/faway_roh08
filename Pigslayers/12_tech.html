⚠ 조치 필요 (3건)

1. Discord 웹훅 4건이 토큰까지 평문 하드코딩 — global-metadata.dat에 포함. 웹훅 URL은 그 자체가 인증 수단이라, APK를 가진 누구나 해당 채널에 임의 메시지를 보낼 수 있습니다. 즉시 재발급(revoke) 권장. 가장 시급한 항목입니다.

2. AdMob 공식 테스트 광고 유닛 ID 잔존 (ca-app-pub-3940256099942544/…)이 실 유닛과 함께 존재. IL2CPP 특성상 런타임 실사용 여부는 정적 분석으로 확정 불가하나, 실제로 쓰인다면 리워드 광고 수익이 0이 됩니다. 코드 확인 필요.

3. 개발용 산출물 동봉 — google-services-desktop.json(데스크톱용), Firebase App Check Debug Registrar가 릴리스에 포함, SRDebugger 포함.

어트리뷰션 — 부재

AppsFlyer / Adjust / Facebook SDK가 전혀 없습니다. 현재는 Google Play Install Referrer + GA4 조합으로 추정됩니다. 유료 UA 캠페인을 집행할 계획이라면 어트리뷰션 SDK 도입이 필요합니다 — 사후 도입은 초기 코호트 데이터를 영구히 잃습니다.
